## About

This is the public support repository for Carnatic Metronome, a professional metronome and accompaniment app for Carnatic music practice.

The main development repository is private, but this repository is public so users can report issues and request features without needing access to the codebase.

## Features

Carnatic Metronome provides:
- Multiple Carnatic tala patterns (Adi, Rupaka, Misra Chapu, Khanda Chapu, and more)
- Flexible nadai options (Chatusra, Tisra, Khanda, Misra, Sankeerna)
- Shruthi (pitch) generator with multiple pitch options
- High-quality mridangam accompaniment samples
- Adjustable tempo (20-200 BPM)
- Visual beat indicators and cycle counting
